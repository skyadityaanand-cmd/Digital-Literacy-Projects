Digital Literacy Portfolio 
Student Digital Ambassador Initiative 
This repository contains my complete Digital Literacy Portfolio, created as part of my role as 
a Student Digital Ambassador. The goal of this project is to help peers navigate the digital 
world safely and professionally. It includes resources on digital awareness, professional 
portfolio setup, collaboration tools, email etiquette, and cybercrime prevention.

Project Modules 
Task 1: Digital Literacy Awareness Infographic Summary: Designed a visual infographic to 
help peers understand digital literacy, safe internet practices, and professional online 
presence. Details: Created using Canva, the infographic highlights the definition of digital 
literacy, essential safe internet practices like using strong passwords and recognizing 
phishing, and the importance of maintaining a professional online presence. 
Task 2: Build Your Student Digital Portfolio Summary: Established a foundational professional 
online presence across key platforms used by employers and the academic community. 
Platform Links: GitHub Profile LinkedIn Profile Stack Overflow Profile Details: GitHub is 
utilized for version control and showcasing coding projects, LinkedIn for professional 
networking and brand building, and Kaggle for honing skills in machine learning and data 
science. 
Task 3: Explore Coding & Collaboration Platforms Summary: Explored coding practice 
environments and built a functional collaboration tool to assess digital literacy. Deliverable 
Links: HackerRank Profile Digital Literacy Awareness Quiz Details: 
Completed introductory Python challenges on HackerRank to grasp basic syntax and logic 
f
low. Utilized Google Workspace to design a 5-question quiz via Google Forms, which 
automatically collected responses into a linked Google Sheet. 
Task 4: Professional Email & Etiquette Guide Summary: Drafted sample professional emails 
for academic and internship contexts, and created a Do's and Don'ts checklist for social 
media. Details: This section emphasizes the "pause rule" before posting online and highlights 
the importance of keeping frustrations private to maintain a professional reputation. 
Separating professional networking accounts from personal profiles is also recommended to 
maintain clear boundaries. (Note: See the task-4-email-etiquette/ folder for email drafts and 
the social media checklist). 
Task 5: Cybercrime Awareness & Prevention Guide Summary: Researched a real-world 
cybercrime scenario (UPI Fraud) and created an actionable prevention checklist tailored for 
Indian college students. Details: The research highlights how modern cybercriminals rely 
heavily on psychological manipulation and social engineering—such as creating panic or 
greed—to access accounts via screen-sharing apps. It also emphasizes the importance of 
avoiding public Wi-Fi networks for financial transactions or accessing sensitive personal data. 
(Note: See the task-5-cybercrime/ folder for the case study and prevention checklist).
